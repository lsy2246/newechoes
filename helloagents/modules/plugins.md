# plugins

## 职责

构建/内容处理插件（src/plugins），用于扩展 Astro/Markdown 处理流程（待确认）。包含开发工具脚本（如 `new-post.mjs`），用于创建文章内容。

## 接口定义（可选）

无对外公共 API。

## 行为规范

### 构建时处理
**条件**: 构建或预览执行。
**行为**: 插件按约定处理内容或页面。
**结果**: 输出符合预期的页面/资源。

### 开发脚本
**条件**: 执行 `pnpm new-post` 创建文章。
**行为**: `new-post.mjs` 生成 `src/content` 下的 Markdown 文件。
**结果**: 新文章可被内容系统识别。

## 依赖关系

```yaml
依赖: config（构建配置，待确认）
被依赖: 构建流程
```
